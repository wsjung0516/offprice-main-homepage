Summary

This program is a website for clothing closeout sales. It enables sellers to sell their inventory in bulk quantities. Wholesale or retail businesses can use this website to select and purchase items. The program allows users to store selected items in a separate cart and initiate transactions with the sellers by verifying the seller's information provided with the product. The program focuses on the following considerations:

  • Fast search: Ensuring users can quickly find their desired items among a large number of products.
  • Easy product registration for sellers: Designing a user-friendly interface for product registration without the need for extensive instructions.
  • Responsive mode: Applying responsive design to enable easy product registration and browsing on any hardware device. It particularly allows direct photo uploads from mobile devices.
